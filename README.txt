INSTALLATION:
	1. Downlaod the entire `twisty` folder to the `/wp-content/themes/` directory
	2. Activate the Theme through the 'Appearance / themes ' menu in WordPress
	
FREQUENTLY ASKED QUESTIONS:
	Does this work work with newest WP version and also older versions? 	
		Yes, this theme works really fine with WordPress 7.4.0. It also works great with WP 8.0.8
